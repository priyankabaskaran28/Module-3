## List Operations in Python: Sum of List Items
🎯 Aim

To write a Python program that calculates the sum of all elements in a list.

🧠 Algorithm

Define a list of numbers.
Use Python’s built-in sum() function to calculate the total.
Print the result.

🧾 Program
```
list1=[1, 2, -8]
total=sum(list1)
print(total)
```
